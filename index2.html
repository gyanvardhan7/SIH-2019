what Syzygy is all about

The World Wide Web is a host to a wide range of criminal activities such as spam-advertised e-commerce, financial fraud, and malware dissemination. Therefore, systems which can detect such malicious web pages must be developed.
Malware detection is an important factor in the security of computer systems. However, currently utilized signature-based methods cannot provide accurate detection of zero-day attacks and polymorphic viruses. That is why the need for machine learning-based detection arises. Our model examines a document at multiple levels of granularity and is capable of profiling maliciousness at each level. Moreover, the model uses shared weights for analysis at each level providing the efficiency needed for a scalable Real-time system.
